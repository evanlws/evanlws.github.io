Musicality

Musicality - Never Miss a Beat is among one of my most ambitious projects to date, yet the concept is very simple. It is an app that scans the user's music library and allows the user to be notified whenever an artist releases new music. I created this app because I felt I was always missing out on new music by my favorite artists. There are so many newer artists in the music scene, it's hard to keep up!

Written in Objective-C, I've used a variety of different tools and frameworks for this iOS App. Musicality first uses the built in MPMediaLibrary to get the users song information, then the iTunes Search API to match the library with the items in iTunes. It also fetches and parses data JSON and XML using the NSXMLParser. The app also uses Crashlytics for crash reports, part of Twitter's Fabric framework. The UI was built using UIKit, storyboards and xib files.

Tomato Ticker

Tomato Ticker: The Pomodoro Timer was created in order to improve productivity, without being obtrusive. My philosophies are to create apps that I myself would want to use. Tomato Ticker does exactly that. It was originally an app I created for myself, to help focus on the task at hand and increase my overall productivity. It's a very simple but extremely effective Mac OS X App.

Written in Objective-C, the pomodoro timer is a small app that utilizes the system menu bar and NSTimer.

Juggler

Juggler: A Game About Juggling is a small iOS arcade game. Originally developed in order for me to learn SpriteKit, it quickly evolved to a full game on the App Store.

Written in Objective-C, Juggler is built using SpriteKit. It utilizes Chartboost and iAD for ad integration and is also compatible with GameCenter.

Top Marks
In Progress

Top Marks was an iOS app I created because I wanted an easy way to keep track of my grades. The biggest challenge was designing the user experience. I wanted it to be simple but compatible with a variety of education institutions.

I originally began working on this in Objective-C, but I since decided it would be good practice to try and create Top Marks 100% using Swift. I'm using UIKit storyboards and xib files for the UI. CoreData is used for the backend, with CloudKit integration.
